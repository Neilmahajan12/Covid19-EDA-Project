# EDA Project on Covid-19 Dataset

This repository contains an Exploratory Data Analysis (EDA) project performed on Covid-19 Dataset. The analysis is done using Python in a Google Colab notebook, and the visualizations are created using Power BI.

## Files
- `covid19 EDA_Project.ipynb`: The Jupyter notebook containing the EDA.
- `covid19 Visualization.pbix`: The Power BI file with the visualizations.

## How to Use
1. Open the `covid19 EDA_Project.ipynb` file in Google Colab.
2. Run the cells to perform the analysis.
3. Open the `covid19 Visualization.pbix` file in Power BI Desktop to view the visualizations.

## Requirements
- Python
- Google Colab
- Power BI Desktop
